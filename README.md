Falling Sphere Simulation

A simple Python simulation that demonstrates the motion of a falling sphere under gravity.

📌 Description

This project simulates the motion of a sphere falling due to gravity using Python.
It calculates the position and velocity over time and visualizes the motion using a graph.

⚙️ Technologies Used

- Python
- NumPy
- Matplotlib

🚀 How to Run

1. Install Python on your system.
2. Install the required libraries:

pip install numpy matplotlib

3. Run the script:

python main.py

📊 Output

The program generates a graph showing the motion of the falling sphere over time.
